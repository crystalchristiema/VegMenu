🌿 VegMenu – Icon Generator

Generate the three PNG icons required by the Chrome extension.

Setup Instructions

  1. Click Generate & Preview Icons above.
  2. Download each icon using the button beneath it.
  3. Place the downloaded files inside the icons/ folder in your VegMenu extension directory:
    icons/icon16.png, icons/icon48.png, icons/icon128.png
  4. Open Chrome → chrome://extensions, enable Developer mode, click Load unpacked, and select the VegMenu folder.

Note: the extension loads fine without custom icons (Chrome shows a default puzzle icon), so you can skip this step while developing.
